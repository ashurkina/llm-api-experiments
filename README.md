# LLM API Experiments

A collection of practical experiments with modern Large Language Model (LLM) APIs, including OpenAI, Anthropic Claude, and Google Gemini.

This repository serves as a hands-on playground for exploring prompt engineering, structured outputs, tool calling, evaluation techniques, and other common LLM application patterns.

## Goals

* Learn and compare different LLM providers and APIs
* Explore prompt engineering techniques
* Experiment with structured outputs and tool calling
* Build practical examples of LLM-powered workflows
* Evaluate model behavior across different scenarios

## Technologies

* Python
* OpenAI API
* Anthropic API
* Google Gemini API
* Jupyter Notebooks
* Pandas
